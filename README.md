# FreeDom Services — Site React

## Installation
1. Remplacez le contenu de votre dossier `freedom-services` par les fichiers de cette archive.
2. Dans VS Code, ouvrez le terminal dans ce dossier.
3. Lancez :
   npm install
   npm run dev
